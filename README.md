# Admin-Finder

Find WebSite Admin Panel in Few Minutes!

Run:
---------
```
$ git clone https://github.com/Ashkan-Z9/Admin-Finder
$ cd Admin-Finder
$ python3 AdminFinder.py

```
Usage:
---------
Just Enter Target URL with slash(/) end of URL.
<br/>
example: https://www.github.com/
